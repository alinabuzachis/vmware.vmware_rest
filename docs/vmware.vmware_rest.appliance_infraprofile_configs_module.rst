*************************************************
vmware.vmware_rest.appliance_infraprofile_configs
*************************************************

**Exports the desired profile specification.**


Version added: 2.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Exports the desired profile specification.



Requirements
------------
The below requirements are needed on the host that executes this module.

- vSphere 7.0.3 or greater
- python >= 3.6
- aiohttp

Examples

        state: export
        profiles:
        - ApplianceManagement
      register: result
